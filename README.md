# 📊 Executive Sales Dashboard

🚀 Data Analytics Project | Tableau Dashboard | Business Insights

---

## 📌 Overview
This project showcases an interactive Executive Sales Dashboard built using Tableau to analyze sales performance and key business metrics.

---

## 🎯 Objective
To provide actionable insights into sales trends, revenue growth, and regional performance to support business decision-making.

---

## 📊 Dashboard Features
- Total Sales & Profit Overview  
- Region-wise Sales Analysis  
- Category-wise Performance  
- Monthly & Yearly Trends  
- KPI Indicators (Revenue, Profit, Growth)  

---

## 🛠️ Tools & Technologies
- Tableau  
- Excel / CSV Dataset  

---

## 📈 Key Insights
- Identified top-performing regions  
- Analyzed seasonal sales patterns  
- Highlighted high-profit categories  
- Compared revenue trends over time  

---


---

## ▶️ How to Use
1. Download the `.twbx` / `.twb` file  
2. Open in Tableau  
3. Use filters to explore insights
